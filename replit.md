# Boudi For Tech - Personal Website

## Overview
A personal portfolio website showcasing web development projects and skills. This is a static HTML/CSS/JavaScript website with a modern, responsive design featuring dark/light theme toggle.

## Project Structure
- `index.html` - Main HTML file
- `css/` - Stylesheets (custom styles, normalize, Font Awesome)
- `js/` - JavaScript for interactivity (theme toggle, menu, animations)
- `Images/` - Profile images and project screenshots
- `webfonts/` - Font Awesome icon fonts

## Features
- Responsive design
- Dark/light theme toggle
- Animated sidebar navigation
- Hero section with typing animation
- About section with skills showcase
- Projects showcase
- Contact form (using Web3Forms API)

## Technical Stack
- HTML5
- CSS3 (custom properties, animations)
- Vanilla JavaScript
- Font Awesome icons
- Google Fonts (Montserrat)

## Running Locally
This is a static website that can be served with any HTTP server on port 5000.
